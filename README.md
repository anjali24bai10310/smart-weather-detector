# smart-weather-detector

Hi there! I created a fantastic little Java program called Smart Weather Detector to provide you with quick updates on weather without much hassle right from your computer! It’s great for people who are just learning Java, as well as those who want to see how real-life applications are developed using simple coding techniques.

What’s the Big Idea?
Imagine that you input your city into this program, and boom – there’s your weather report in an instant so that you can decide if you should wear a jacket or if shorts are the go! Nice and simple, right? That’s the aim of this application – to allow you to make smart choices easier to make because of some code magic.

What Does It Do?
It gives you the type of weather (e.g., sunny, rainy – whatever!) and temperature for the specified location based on the input.
It will allow you to have dialogue with your application through the console.
It is fast – lightning fast – with no bloat issues.
It is clean – the code is organized into logical, modular classes.
Technical Elements
Pure Java!
OOP – which means classes, objects, etc.

If/Else 
    logic for performing actions based on the weather.
Methods to keep everything tidy.
Exceptional handling so the program won’t crash.
